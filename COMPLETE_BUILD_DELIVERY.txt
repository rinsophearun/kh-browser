HOW TO BUILD (THREE OPTIONS):
═════════════════════════════════════════════════════════════════════════════

OPTION 1: Universal BUILD.py (Recommended)
──────────────────────────────────────────
Works on any platform for any target:

   macOS:
   $ python BUILD.py macos    # Creates .app and .dmg

   Windows:
   > python BUILD.py windows  # Creates .exe and Setup.exe

   Both:
   $ python BUILD.py all      # Creates everything

Advantages:
   ✓ Single source of truth
   ✓ Works on all platforms
   ✓ Automated verification
   ✓ Professional output


OPTION 2: Windows Quick Build (build_windows_installer.bat)
────────────────────────────────────────────────────────────
Direct Windows build without venv:

   > build_windows_installer.bat

Advantages:
   ✓ No venv setup needed
   ✓ Quick one-click build
   ✓ Opens output folder automatically
   ✓ Good for rapid testing


OPTION 3: Windows Package (WINDOWS_BUILD_PACKAGE/)
───────────────────────────────────────────────────
Dedicated Windows build tools:

   Command Prompt:
   > BUILD.bat

   PowerShell:
   > BUILD.ps1

Advantages:
   ✓ Standalone scripts
   ✓ No root directory needed
   ✓ Detailed setup wizard
   ✓ Complete troubleshooting

BUILD PROCESS:
═════════════════════════════════════════════════════════════════════════════

What BUILD.py Automatically Does:

1. VERIFICATION (< 1 minute)
   ✓ Checks all source files
   ✓ Verifies icons present
   ✓ Confirms assets available

2. CLEANUP (< 1 minute)
   ✓ Removes previous builds
   ✓ Clears cache directories
   ✓ Clean slate

3. DEPENDENCIES (3-5 minutes)
   ✓ Creates virtual environment (if needed)
   ✓ Installs all Python packages
   ✓ Verifies installation

4. BUILD APPLICATION (5-10 minutes)
   ✓ PyInstaller bundles Python
   ✓ Includes all dependencies
   ✓ Bundles all resources
   ✓ Creates native executable

5. CREATE INSTALLER (2-5 minutes)
   ✓ macOS: Creates DMG image
   ✓ Windows: Runs Inno Setup
   ✓ Creates professional installers

6. VERIFICATION (< 1 minute)
   ✓ Checks all outputs created
   ✓ Verifies file integrity
   ✓ Reports success

TOTAL TIME:
   First build: 15-25 minutes
   Subsequent builds: 10-15 minutes
